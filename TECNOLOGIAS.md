# 🔬 Impacto por tecnologia (por área)

Cada tecnologia não é uma escolha estética — ela produz um **efeito concreto** em **Software**, **Infraestrutura**, **Custo** e **Segurança/LGPD**. Abaixo, o porquê de cada uma.

![software](assets/sections/software.png)

## .NET 10 / ASP.NET Core (C#)
- **Software:** linguagem tipada e madura, alto desempenho, ecossistema completo e testes de primeira classe → menos bugs, evolução segura.
- **Infra:** servidor Kestrel leve e **multiplataforma**; roda em contêiner Linux, não exige mais Windows/IIS.
- **Custo:** open-source e gratuito; **mais requisições com menos CPU/RAM** que o legado → instância menor, fatura menor.
- **Segurança/LGPD:** framework **ativamente mantido** (patches contínuos), autenticação moderna (JWT), validação integrada.

## Python + FastAPI
- **Software:** altíssima **produtividade** e legibilidade; APIs validadas por tipos e **autodocumentadas**; assíncrono por padrão.
- **Infra:** leve, ideal para contêiner; sobe rápido e escala horizontalmente.
- **Custo:** gratuito e com **mão de obra abundante** (contratar/treinar é barato) — o oposto do ColdFusion.
- **Segurança/LGPD:** validação rigorosa de entrada (Pydantic) reduz classes inteiras de erro; dependências mantidas.

## React + TypeScript + Vite
- **Software:** interfaces **tipadas** (menos erro em produção), componentizadas e testáveis; build instantâneo (Vite).
- **Infra:** entrega estática, cacheável, servida por qualquer CDN/servidor — barata e rápida.
- **Custo:** ecossistema gratuito e **o mais contratável do mercado**.
- **Segurança/LGPD:** renderização que escapa dados por padrão (mitiga XSS), foco/acessibilidade nativos.

![infra](assets/sections/infra.png)

## Docker (contêineres)
- **Software:** ambiente **idêntico** em dev, teste e produção → fim do "na minha máquina funciona".
- **Infra:** **consolidação real** — vários sistemas dividem um único host (mais densidade, menos servidores); deploy e rollback previsíveis.
- **Custo:** **menos VMs** = menos compute e menos manutenção; melhor uso do hardware.
- **Segurança/LGPD:** isolamento entre serviços, superfície reduzida, imagens versionadas e auditáveis.

## Linux / Oracle Linux + Oracle Cloud (OCI)
- **Software:** base estável e padrão de mercado para serviços de longa duração.
- **Infra:** **suporte até 2032**, footprint enxuto, pronto para automação e escala; shapes ARM/Ampere disponíveis.
- **Custo:** **licença de SO R$ 0** (fim de Windows Server + CALs); instâncias mais baratas e dimensionáveis.
- **Segurança/LGPD:** patches contínuos, **SELinux** (controle de acesso mandatório), hardening que o mundo Windows legado não tinha.

## Terraform (Infraestrutura como Código)
- **Software:** a infraestrutura vira **código versionado e revisável**.
- **Infra:** ambientes **reproduzíveis** em minutos; nada de servidor "artesanal" que ninguém sabe recriar.
- **Custo:** provisiona só o necessário; destrói o que não usa → **sem desperdício**.
- **Segurança/LGPD:** configuração **auditável** e consistente; recuperação de desastre muito mais rápida.

## PostgreSQL (roadmap de dados)
- **Software:** banco relacional robusto, padrão e com recursos avançados.
- **Infra:** roda em Linux/contêiner; escala e replica bem.
- **Custo:** **open-source, licença R$ 0** — caminho para eliminar também o custo de licença de banco.
- **Segurança/LGPD:** criptografia, controle de acesso fino e comunidade que corrige rápido.

## Motor de documentos: PyMuPDF · Chromium · LibreOffice
- **Software:** substitui componentes proprietários de PDF por bibliotecas abertas e testáveis.
- **Infra:** roda em Linux/contêiner, sem DLLs presas ao Windows (**fim do lock-in**).
- **Custo:** **sem licença** de geração de documentos.
- **Segurança/LGPD:** cadeia de geração controlada e reproduzível (crítico para atos oficiais com valor legal).

![segurança e LGPD](assets/sections/lgpd.png)

## Segurança: OWASP CRS · Wazuh · SELinux · Let's Encrypt · JWT · fail2ban
- **Software:** autenticação por token (JWT) e boas práticas embutidas na aplicação.
- **Infra:** **WAF** na borda (OWASP Top 10), **SIEM/XDR** (Wazuh) para detecção e resposta, **fail2ban** contra abuso, **TLS automático** (Let's Encrypt).
- **Custo:** **todas essas camadas são open-source — licença R$ 0** (equivalentes comerciais custariam caro).
- **Segurança/LGPD:** **defesa em profundidade** + **trilha de evidências** para auditoria → sustenta a defesa de "medidas de segurança adequadas" exigida pela LGPD.

## Qualidade: Cypress · pytest · xUnit
- **Software:** testes automatizados que **bloqueiam o deploy** se algo quebra → menos regressão.
- **Infra:** integrados ao CI; cada mudança é verificada antes de ir ao ar.
- **Custo:** **menos incidentes = menos horas apagando incêndio** (manutenção mais barata).
- **Segurança/LGPD:** validação contínua reduz falhas que expõem dados.
